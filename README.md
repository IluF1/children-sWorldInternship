## Install

To get started:

1. Clone the repository:
```sh
git clone https://github.com/IluF1/children-sWorldInternship.git
cd my-app
```

2. Install all dependencies in the project:
```sh
yarn
```

3. Run project:
```sh
yarn run dev
```
